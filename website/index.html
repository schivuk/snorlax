We are Team Snorlax

A real legit smart bed.

Concept

Snorlax is a smart mattress pad aimed at allowing people to measure their sleeping patterns. The mattress pad consists of a network of pressure sensors inside the mattress and a couple of accelerometers to measure the position and movements of the user, a WiFi module to communicate this information to a web server, and a web application that allows users to login, register their device, and view their sleep patterns when they wake up. Users can set an alarm time via the web application, and the smart mattress will wake the user up at an optimal time based on estimated sleep cycle detection.

Competitive Analysis

Our primary competitor is Eight Sleep. Eight is a smart mattress pad that does all of the sleep tracking without the need of a wearable device. However, according to their website, Eight has the following features: Heating pads to pre-warm the mattress pad Ability to communicate with Nest to cool the room Ability to monitor heart rate, breathing rate and noise levels Ability to control bedroom lighting

Sleep as Android is a sleep tracking android app that determines the user’s optimal waking point through the use of accelerometers. The problem with Sleep as Android is you need to put your phone on your bed in order to get any readings. You run the risk of dropping the phone off the bed while sleeping, or receiving extra exposure to EMF radiation by keeping your phone next to your head. Regardless, it is amazing how they are able to get accurate data just through the phone’s accelerometer data. Our goal is leverage their experience and technology into our own smart bed design.

Motivation

The incentive for this project was to have an entirely non-intrusive system (including wearables), that augments the functionality of current mobile phone applications. By spanning the entire mattress with sensor coverage and having multiple accelerometers at different parts of the mattress, our goal is to broaden the sensor data that phones can collect and have more accurate sleep information.

Design Requirements

  • General: The system shall consist of a network of sensors that are embedded into a mattress pad, so the product can be used by placing it on top of any mattress. The product shall be used to track body positions/movements during sleep, to control an alarm, and to allow users to view sleep data over the internet.
  • Hardware components: The main hardware components shall involve a network of pressure sensors, at least 2 accelerometers, a microphone, an MCU to measure sensor data, a WiFi module to transmit the data to a web server, an amplifier-speaker system to sound alarms, and a computer to store and analyse the data.
  • Non-invasiveness: The Snorlax mattress pad shall be entirely non-invasive by having a design such that the sensor network shall be located on the bottom of the mattress pad, conductive thread shall be used for wiring the sensor network and the user will have no wearable component.
  • Application interfacing: The user shall be able to register their smart mattress pad to their preferred WiFi network using their phone (by entering in the network name and password). The mattress pad will then connect to the specified WiFi network and communicate / log data to the web server over the internet.
  • Authentication: The web server should check for authentication when receiving data from the smart mattress pad, to ensure that arbitrary external requests do not modify logging data.
  • Memory footprint: Logging sensor data at very frequent intervals can lead to an arbitrarily large memory load for the server. Measures must be taken to log only important information such as position changes, reading spikes from the accelerometer or microphone.
  • Alarm: The smart mattress pad shall have an alarm feature, whereby users can set an alarm time through the web application. The smart mattress pad shall find an optimal time for the user to wake up based on estimated sleep cycle detection, and sound the alarm at the optimal time within a 30 minute window of the specified time. The smart mattress pad shall detect if the user is still on the bed, and only turn off the alarm after the user gets off the bed.
